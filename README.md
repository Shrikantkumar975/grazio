# Grazio — Intelligent Livestock Tracking

A robust Laravel 12 application designed for managing and tracking livestock using MongoDB and real-time communication.

## Tech Stack
- Laravel 12
- MongoDB (`mongodb/laravel-mongodb`)
- Alpine.js & Tailwind CSS (Zenith Theme)

## Setup
1. `composer install`
2. `npm install && npm run build`
3. Copy `.env.example` to `.env` and configure MongoDB
4. `php artisan key:generate`
5. `php artisan serve`
